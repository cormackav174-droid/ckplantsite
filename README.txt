CK Plant Ltd website package.

How to use with Vercel:
1. Unzip this package on your computer.
2. Create a new GitHub repository named ckplant-site.
3. Upload ALL the unzipped files and folders to the repository.
4. In Vercel, click Add New > Project.
5. Import the GitHub repository.
6. Click Deploy.
7. In Vercel project settings, add domain ckplant.ie.
